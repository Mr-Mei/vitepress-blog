一、水平居中 ​

1. 使用text-align属性 ​

父元素设置 text-align: center，子元素设置 display: inline-block，实现水平居中

html
<!-- 公共html结构 -->
<div class="parent">
  <div class="child"></div>
</div>
<style>
  * {
    margin: 0;
    padding: 0;
  }
  :root {
    --c-size: 300px;
    --c-bgcolor: #ff8787;
  }
  .parent {
    height: var(--c-size);
    background-color: var(--c-bgcolor);
    /* 主要代码 */
    text-align: center;
  }
  .child {
    width: var(--c-size);
    height: var(--c-size);
    background-color: aqua;
    /* 主要代码 */
    display: inline-block;
  }
</style>

2. 定宽块级元素水平居中 ​

html
<!-- 公共html结构 -->
<div class="parent">
  <div class="child"></div>
</div>
<!--方法1 子元素设置 margin: 0 auto; -->
<style>
  * {
    margin: 0;
    padding: 0;
  }
  :root {
    --c-size: 300px;
    --c-bgcolor: #ff8787;
  }
  .parent {
    height: var(--c-size);
    background-color: var(--c-bgcolor);
  }
  .child {
    height: var(--c-size);
    background-color: aqua;
    /* 主要代码 */
    width: var(--c-size);
    margin: 0 auto;
  }
</style>
<!--方法2 子元素设置 position: relative; 通过 left 属性 和 margin 实现 -->
<style>
  * {
    margin: 0;
    padding: 0;
  }
  :root {
    --c-size: 300px;
    --c-bgcolor: #ff8787;
  }
  .parent {
    height: var(--c-size);
    background-color: var(--c-bgcolor);
  }
  .child {
    height: var(--c-size);
    background-color: aqua;
    /* 主要代码 */
    width: var(--c-size);
    position: relative;
    left: 50%;
    /* margin-left 为 负的宽度的一半 */
    margin-left: -150px;
  }
</style>
<!--方法3 当元素开启绝对定位或者固定定位时， left 属性和 right 属性一起设置就会拉伸元素的宽度，在配合 width 属性与 margin 属性就可以实现水平居中 -->
<style>
  * {
    margin: 0;
    padding: 0;
  }
  :root {
    --c-size: 300px;
    --c-bgcolor: #ff8787;
  }
  .parent {
    height: var(--c-size);
    background-color: var(--c-bgcolor);
    /* 主要代码 */
    position: relative;
  }
  .child {
    height: var(--c-size);
    background-color: aqua;
    /* 主要代码 */
    position: absolute;
    /* 水平拉满屏幕 */
    left: 0;
    right: 0;
    /* 拉满屏幕之后设置宽度，最后通过 margin: auto; 实现水平居中 */
    width: var(--c-size);
    margin: auto;
  }
</style>
<!--方法4 当元素开启绝对定位或者固定定位时， left 属性和 transform 属性即可实现水平居中 -->
<style>
  * {
    margin: 0;
    padding: 0;
  }
  :root {
    --c-size: 300px;
    --c-bgcolor: #ff8787;
  }
  .parent {
    height: var(--c-size);
    background-color: var(--c-bgcolor);
    /* 主要代码 */
    position: relative;
  }
  .child {
    width: var(--c-size);
    height: var(--c-size);
    background-color: aqua;
    /* 主要代码 */
    position: absolute;
    left: 50%;
    transform: translateX(-50%);
  }
</style>

二、垂直居中 ​

1. 行内块级元素垂直居中 ​

若元素是行内块级元素，基本思想是子元素使用display: inline-block、vertical-align: middle，并让父元素行高等同于高度。

html
<!-- 公共html结构 -->
<div class="parent">
  <div class="child"></div>
</div>
<style>
  * {
    margin: 0;
    padding: 0;
  }

  :root {
    --c-size: 300px;
    --c-bgcolor: #ff8787;
  }

  .parent {
    width: var(--c-size);
    background-color: var(--c-bgcolor);
    height: 600px;
    /* 为父级容器设置行高 */
    line-height: 600px;
  }

  .child {
    width: var(--c-size);
    height: var(--c-size);
    background-color: aqua;
    /* 将子级元素设置为 inline-block 元素 */
    display: inline-block;
    /* 通过 vertical-align: middle; 实现居中 */
    vertical-align: middle;
  }
</style>

html
<!-- 公共html结构 -->
<div class="parent">
  <div class="child"></div>
</div>
<!-- 通过top: 50%; margin-top: 等于负的高度的一半就可以实现垂直居中 -->

    background-color: aqua;
    /* 主要代码 */

    margin-top: -150px;
  }
</style>
<!-- 通过top 和 bottom 将子元素拉伸至100%，设置指定的高度，通过margin:auto;即可实现垂直居中 -->

    margin: 0;
    padding: 0;
  }

  :root {
    --c-size: 300px;

<!-- 通过top 配合 tansform 即可实现垂直居中，适用于多种场合 -->

  }

三、水平垂直居中 ​

实现水平垂直居中布局基本就是将水平居中布局与垂直居中布局结合起来。

1. 行内块级实现水平垂直居中 ​

html
<!-- 公共html结构 -->
<div class="parent">
  <div class="child"></div>
</div>
<!-- 容器元素行高等于容器高度，通过 text-align: center实现水平居中，将子级元素设置为水平块级元素，通过 vertical-align:middle实现垂直居中 -->
<style>
  * {
    margin: 0;
    padding: 0;
  }

  :root {
    --c-size: 300px;
    --c-bgcolor: #ff8787;
  }

  .parent {
    width: calc(var(--c-size) * 5);
    background-color: var(--c-bgcolor);
    height: calc(var(--c-size) * 2);
    /* 1. 设置行高等于容器高度 */
    line-height: calc(var(--c-size) * 2);
    /* 通过 text-align: center; 实现水平居中 */
    text-align: center;
  }

  .child {
    width: var(--c-size);
    height: var(--c-size);
    background-color: aqua;
    /* 将子级元素设置为水平块级元素 */
    display: inline-block;
    /* 通过 vertical-align: middle; 实现垂直居中 */
    vertical-align: middle;
  }
</style>

2. 定位实现水平垂直居中 ​

html
<!-- 公共html结构 -->
<div class="parent">
  <div class="child"></div>
</div>
<!-- 方案一、父元素开启相对定位，子元素开启绝对定位，设置该元素的偏移量值为50%减去宽度/高度的一半 -->
<style>
  * {
    margin: 0;
    padding: 0;
  }

  :root {
    --c-size: 300px;
    --c-bgcolor: #ff8787;
  }

  .parent {
    width: calc(var(--c-size) * 5);
    background-color: var(--c-bgcolor);
    height: calc(var(--c-size) * 2);
    /* 开启相对定位 */
    position: relative;
  }

  .child {
    width: var(--c-size);
    height: var(--c-size);
    background-color: aqua;
    /* 开启绝对定位 */
    position: absolute;
    /* 设置该元素的偏移量，值为 50%减去宽度/高度的一半 */
    left: calc(50% - 150px);
    top: calc(50% - 150px);
  }
</style>
<!-- 方案二、父元素开启相对定位，子元素开启绝对定位，设置该元素的偏移量值为50%，通过外边距负值的方式将元素移动至居中 -->
<style>
  * {
    margin: 0;
    padding: 0;
  }

  :root {
    --c-size: 300px;
    --c-bgcolor: #ff8787;
  }

  .parent {
    width: calc(var(--c-size) * 5);
    background-color: var(--c-bgcolor);
    height: calc(var(--c-size) * 2);
    /* 开启相对定位 */
    position: relative;
  }

  .child {
    width: var(--c-size);
    height: var(--c-size);
    background-color: aqua;
    /* 开启绝对定位 */
    position: absolute;
    /* 设置该元素的偏移量，值为 50% */
    left: 50%;
    top: 50%;
    margin-left: -150px;
    margin-top: -150px;
  }
</style>
<!-- 方案三、父元素开启相对定位，子元素开启绝对定位，将子元素拉满整个容器，通过 margin: auto 实现水平垂直居中 -->
<style>
  * {
    margin: 0;
    padding: 0;
  }

  :root {
    --c-size: 300px;
    --c-bgcolor: #ff8787;
  }

  .parent {
    width: calc(var(--c-size) * 5);
    background-color: var(--c-bgcolor);
    height: calc(var(--c-size) * 2);
    /* 开启相对定位 */
    position: relative;
  }

  .child {
    width: var(--c-size);
    height: var(--c-size);
    background-color: aqua;
    /* 开启绝对定位 */
    position: absolute;
    /* 将子元素拉满整个容器 */
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    /* 通过 margin:auto 实现水平垂直居中 */
    margin: auto;
  }
</style>
<!-- 方案四、父元素开启相对定位，子元素开启绝对定位，设置该元素的偏移量，值为50%，通过 translate 反向偏移的方式，实现居中 -->
<style>
  * {
    margin: 0;
    padding: 0;
  }

  :root {
    --c-size: 300px;
    --c-bgcolor: #ff8787;
  }

  .parent {
    width: calc(var(--c-size) * 5);
    background-color: var(--c-bgcolor);
    height: calc(var(--c-size) * 2);
    /* 开启相对定位 */
    position: relative;
  }

  .child {
    width: var(--c-size);
    height: var(--c-size);
    background-color: aqua;
    /* 开启绝对定位 */
    position: absolute;
    /* 设置该元素的偏移量，值为 50%*/
    left: 50%;
    top: 50%;
    /* 通过translate反向偏移的方式，实现居中 */
    transform: translate(-50%, -50%);
  }
</style>

四、两列布局 ​

所谓的两列布局就是一列定宽(也有可能由子元素决定宽度)，一列自适应的布局。

html
<!-- 公共html结构 -->
<div class="container clearfix">
  <div class="left">定宽</div>
  <div class="right">自适应</div>
</div>
<!-- 公共css结构 -->
<style>
  * {
    margin: 0;
    padding: 0;
  }

  :root {
    --c-size: 400px;
    --c-bgcolor: #ff8787;
  }
  .container {
    height: var(--c-size);
    background-color: var(--c-bgcolor);
  }
  .left {
    width: var(--c-size);
    height: var(--c-size);
    line-height: var(--c-size);
    font-size: 50px;
    text-align: center;
    background-color: yellow;
  }
  .right {
    height: var(--c-size);
    line-height: var(--c-size);
    font-size: 50px;
    text-align: center;
    background-color: aqua;
  }
  /* 清除浮动 */
  .clearfix::after {
    content: '';
    display: block;
    height: 0;
    clear: both;
    visibility: hidden;
  }
</style>
<!-- 1. float+calc() 函数完成左列定宽右列自适应 -->
<!-- 左列开启浮动，右列开启浮动，右列宽度为父级 100% 减去左列的宽度-->
<style>
  .left {
    /* 左边列开启浮动 */
    float: left;
  }
  .right {
    /* 右边列开启浮动 */
    float: left;
    /* 宽度减去左列的宽度 */
    width: calc(100% - var(--c-size));
  }
</style>
<!-- 2. float+margin-left完成左列定宽右列自适应 -->
<!-- 左列开启浮动，通过外边距的方式使该容器的左边有左边列容器的宽度的外边距 -->
<style>
  .left {
    /* 左边列开启浮动 */
    float: left;
  }
  .right {
    /* 通过外边距的方式使该容器的左边有var(--c-size)距离 */
    margin-left: var(--c-size);
  }
</style>
<!-- 3. absolute+margin-left完成左列定宽右列自适应 -->
<!-- 开启定位脱离文档流，通过外边距的方式使该容器的左边有左边列容器的宽度的外边距 -->
<style>
  .left {
    /* 开启定位脱离文档流 */
    position: absolute;
  }
  .right {
    /* 通过外边距的方式使该容器的左边有var(--c-size)距离 */
    margin-left: var(--c-size);
  }
</style>
<!-- 4. float+overflow完成左列定宽右列自适应 -->
<!-- 左侧元素开始浮动，右侧自适应元素设置 overflow 会创建一个BFC完成自适应 -->
<style>
  .left {
    /* 左列开启浮动 */
    float: left;
  }
  .right {
    /* 右侧自适应元素设置 overflow 会创建一个BFC 完成自适应 */
    overflow: hidden;
  }
</style>
<!-- 5. 通过Flex方案实现 -->
<style>
  .container {
    display: flex;
  }
  .right {
    /* flex: 1; 表示 flex-grow: 1; 即该项占所有剩余空间 */
    flex: 1;
  }
</style>
<!-- 5. 通过Grid方案实现 -->
<style>
  .container {
    display: grid;
    /* 将其划分为两行，其中一列有本身宽度决定， 一列占剩余宽度*/
    grid-template-columns: auto 1fr;
  }
</style>

五、三列布局 ​

三列布局主要分为2种：

    1. 前两列定宽，最后一列自适应，这一种本质上与两列布局没有什么区别，可以参照两列布局实现。
    1. 前后两列定宽，中间自适应。

html
<!-- 公共html结构 -->
<div class="container clearfix">
  <div class="left">左</div>
  <div class="content">内容</div>
  <div class="right">右</div>
</div>
<!-- 公共css结构 -->
<style>
  * {
    margin: 0;
    padding: 0;
  }

  :root {
    --c-size: 400px;
    --c-bgcolor: #ff8787;
  }
  .container {
    height: var(--c-size);
    background-color: var(--c-bgcolor);
  }
  .left {
    width: var(--c-size);
    background-color: yellow;
  }
  .content {
    background-color: #d9480f;
  }

  .right {
    width: var(--c-size);
    background-color: aqua;
  }
  .left,
  .content,
  .right {
    font-size: 50px;
    height: var(--c-size);
    line-height: var(--c-size);
    text-align: center;
  }
  /* 清除浮动 */
  .clearfix::after {
    content: '';
    display: block;
    height: 0;
    clear: both;
    visibility: hidden;
  }
</style>

<!-- 1. 通过float实现 -->
<!--需要调整HTML结构，调整后如下： -->
<div class="container clearfix">
  <div class="left">左</div>
  <div class="right">右</div>
  <div class="content">内容</div>
</div>
<!-- 方法1：左列容器开启左浮动，右列容器开启右浮动，自适应元素设置overflow会创建一个BFC完成自适应 -->
<style>
  .left {
    /* 1. 左列容器开启左浮动 */
    float: left;
  }
  .content {
    /* 自适应元素设置 overflow 会创建一个BFC 完成自适应 */
    overflow: hidden;
  }
  .right {
    /* 2. 右列容器开启右浮动 */
    float: right;
  }
</style>
<!-- 方法2：左列容器开启左浮动，右列容器开启右浮动，使中间自适应的宽度为父级容器减去两个定宽的列 -->
<style>
  .left {
    /* 1. 左列容器开启左浮动 */
    float: left;
  }
  .content {
    /* 3. 使中间自适应的宽度为父级容器减去两个定宽的列 */
    width: calc(100% - var(--c-size));
  }
  .right {
    /* 2. 右列容器开启右浮动 */
    float: right;
  }
</style>

<!-- 2. 通过position实现 -->
<!-- 左右两列脱离文档流，并通过偏移的方式到达自己的区域，使中间自适应的宽度为父级容器减去两个定宽的列，通过外边距将容器往内缩小 -->
<style>
  .left {
    /* 1. 左右两列脱离文档流，并通过偏移的方式到达自己的区域 */
    position: absolute;
    left: 0;
    top: 0;
  }
  .content {
    /* 2. 使中间自适应的宽度为父级容器减去两个定宽的列 */
    width: calc(100% - var(--c-size));
    /* 3. 通过外边距将容器往内缩小 */
    margin-right: var(--c-size);
    margin-left: var(--c-size);
  }
  .right {
    position: absolute;
    right: 0;
    top: 0;
  }
</style>
<!-- 3. Flex方案 -->
<style>
  .container {
    display: flex;
  }
  .content {
    flex: 1;
    /* flex: 1; 表示 flex-grow: 1; 即该项占所有剩余空间 */
  }
</style>
<!-- 4. Grid方案 -->
<style>
  .container {
    display: grid;
    /* 将其划分为两行，其中一列有本身宽度决定， 一列占剩余宽度*/
    grid-template-columns: auto 1fr auto;
  }
</style>

六、等分布局 ​

等分布局就是将一个容器平均分成几等份，这里以 4 等分为例。

html
<!-- 公共html结构 -->
<div class="container clearfix">
  <div class="item item1"></div>
  <div class="item item2"></div>
  <div class="item item3"></div>
  <div class="item item4"></div>
</div>
<!-- 公共css结构 -->
<style>
  * {
    margin: 0;
    padding: 0;
  }

  :root {
    --c-size: 400px;
    --c-bgcolor: #ff8787;
  }
  .container {
    height: var(--c-size);
  }
  .item {
    height: 100%;
  }
  .item1 {
    background-color: #eccc68;
  }
  .item2 {
    background-color: #a6c1fa;
  }
  .item3 {
    background-color: #fa7d90;
  }
  .item4 {
    background-color: #b0ff70;
  }
  /* 清除浮动 */
  .clearfix::after {
    content: '';
    display: block;
    height: 0;
    clear: both;
    visibility: hidden;
  }
</style>
<!-- 1. 浮动+百分比方式：开启浮动，使每个元素占25%的宽度 -->
<style>
  .item {
    /* 开启浮动，每个元素占 25% 的宽度 */
    width: 25%;
    float: left;
  }
</style>
<!-- 2. 行内块级+百分比方式：需要注意的是行内块级元素有一些类似于边距的几个像素，导致各25%会超出容器 -->
<style>
  .item {
      * 设置每个元素为行内块级元素，每个元素占 24.5% 的宽度 */
      width: 24.5%;
      /* 因为行内块级元素有一些类似于边距的几个像素，导致各占25会超出容器 */
      display: inline-block;
  }
</style>
<!-- 3. Flex方案 -->
<style>
  .container {
    /* 开启 flex 布局 */
    display: flex;
  }
  .item {
    /* 每个元素占相同的宽度 */
    flex: 1;
  }
</style>
<!-- 4. Grid方案 -->
<style>
  .container {
    /* 开启 grid 布局 */
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    /* 使用 repeat 函数生成如下代码 */
    /* grid-template-columns: 1fr 1fr 1fr 1fr; */
  }
</style>

七、Sticky Footer 布局 ​

所谓的 Sticky Footer 布局并不是一种新的前端技术和概念，它就是一种网页布局。如果页面内容不够长时，底部栏就会固定到浏览器的底部；如果足够长时，底部栏就后跟随在内容的后面。

html
<!-- 公共html结构 -->
<div class="main">
  <div class="header">header</div>
  <div class="container">
    <div class="left">left</div>
    <div class="content">content</div>
    <div class="right">right</div>
  </div>
  <div class="footer">footer</div>
</div>
<!-- 公共css结构 -->
<style>
  * {
    margin: 0;
    padding: 0;
  }

  :root {
    --c-size: 400px;
    --c-bgcolor: #ff8787;
  }
  .container {
    height: var(--c-size);
    display: flex;
  }
  .left {
    height: var(--c-size);
    width: 200px;
    background-color: #f759ab;
  }
  .content {
    height: var(--c-size);
    background-color: #52c41a;
    flex: 1;
  }
  .right {
    height: var(--c-size);
    width: 200px;
    background-color: #f759ab;
  }
  .left,
  .content,
  .right {
    font-size: 70px;
    line-height: var(--c-size);
    text-align: center;
  }
  .header {
    height: 100px;
    background-color: #70a1ff;
  }
  .footer {
    height: 100px;
    background-color: #ff7a45;
  }
  .header,
  .footer {
    line-height: 100px;
    font-size: 52px;
    text-align: center;
  }
</style>
<!-- 1. 绝对定位的方式 -->
<!-- 设置最外层容器高度为100%；让子元素元素相对于容器元素进行定位，并设置容器元素最小高度为100%；在中间区域设置padding-bottom为footer的高度 ；底部栏绝对定位，并一直吸附在底部即可实现。 -->
<style>
  /* 1. 设置最外层容器为100% */
  html,
  body {
    height: 100%;
  }
  /* 2. 让子元素元素相对于容器元素进行定位，并设置容器元素最小高度为100% */
  .main {
    position: relative;
    min-height: 100%;
  }
  /* 3. 在中间区域设置 padding-bottom 为footer 的高度 */
  .container {
    padding-bottom: 100px;
  }
  /* 由于开启了绝对定位，宽度成了自适应，设置为100% bottom:0 始终保持底部 */
  .footer {
    position: absolute;
    width: 100%;
    bottom: 0;
  }
</style>
<!-- 2. 使用calc函数实现：中间的容器最少高度为视口宽度的100% - 头部和底部两部分的高度即可完成该功能 -->
<style>
  .container {
    /* 这里的 中间 部分的容器最少为视口宽度的 100% - 头部和底部两部分的高度即可完成该功能 */
    min-height: calc(100vh - 200px);
  }
</style>
<!-- 3. Flex方案：开启 flex 布局，将子元素布局方向修改为垂直排列，设置最小高度为当前视口，使不管中间部分有多高，始终都可以保持在底部
设置中间部分容器高度为自适应 -->
<style>
  .main {
    /* 开启flex布局 */
    display: flex;
    /* 将子元素布局方向修改为垂直排列 */
    flex-flow: column;
    /* 设置最小高度为当前视口，使不管中间部分有多高，始终都可以保持在底部 */
    min-height: 100vh;
  }
  .container {
    /* 设置 中间 部分自适应 */
    flex: 1;
  }
</style>
<!-- 4. Grid方案：开启 grid 布局，设置最小高度为当前视口，使不管中间部分有多高，始终都可以保持在底部-->
<style>
  .main {
    /* 开启grid布局 */
    display: grid;
    grid-template-rows: auto 1fr auto;
    /* 设置最小高度为当前视口，使不管中间部分有多高，始终都可以保持在底部 */
    min-height: 100vh;
  }
</style>

    flex-flow: column;

    /* 如果超出出现滚动条 */
    overflow: auto;
    /* 设置 中间 部分自适应 */
    flex: 1;

    /* 假设容器内有500px的元素 */
    height: 500px;
  }

    /* 开启grid布局 */

    grid-template-rows: auto 1fr auto;
  }
  .content {
    /* 如果超出出现滚动条 */
    overflow: auto;
  }

    /* 假设容器内有500px的元素 */
    height: 500px;
  }
</style>
